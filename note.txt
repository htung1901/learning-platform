 - JWT chia 3 phần:
 + Header : Chứa thông tin mã hóa
 + Payload: Chứa thông tin dữ liệu nhập vào
 + Signature: Chứ ký tạo ra từ SECRET chỉ server biết
 => 3 phần mã hóa thành 1 chuỗi duy nhất => Cần SECRET để xác minh Token hợp lệ
